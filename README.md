## 🌟 Overview / نظرة عامة

<div dir="rtl">

### العربية

**QuickChat** هو تطبيق محمول ذكي يتيح لك مراسلة أي رقم واتساب مباشرة **بدون الحاجة لحفظه** في جهات الاتصال. مثالي لأصحاب الأعمال، سائقي التوصيل، فرق المبيعات، وأي شخص يتواصل مع أرقام جديدة يوميًا!

</div>

### English

**QuickChat** is a smart mobile app that lets you message any WhatsApp number directly **without saving it** to your contacts. Perfect for business owners, delivery drivers, sales teams, and anyone who contacts new numbers daily!

---

## ✨ Features / المميزات

### 🚀 Core Features / المميزات الأساسية

#### English
- 📞 **Direct Messaging**: Open WhatsApp chat with any number instantly
- 🌍 **195+ Countries**: Smart country code selection (Yemen default)
- 💬 **Message Templates**: 10 ready-to-use templates + create your own
- 📝 **Template Manager**: Add, edit, delete custom message templates
- 🕐 **Recent History**: Auto-save recently messaged numbers
- 📱 **Dual WhatsApp Support**: Works with WhatsApp & WhatsApp Business
- 🌙 **Theme Support**: Light, dark, and system themes
- 🌐 **Bilingual**: Full Arabic and English interface with RTL support
- 🔒 **Privacy First**: No data collection, all local storage
- 🚫 **Ad-Free**: Completely free with no advertisements
- ⚡ **Lightweight**: Under 50MB, fast and efficient

#### العربية
- 📞 **مراسلة مباشرة**: افتح محادثة واتساب مع أي رقم فورًا
- 🌍 **أكثر من 195 دولة**: اختيار ذكي لرمز الدولة (اليمن افتراضيًا)
- 💬 **قوالب الرسائل**: 10 قوالب جاهزة + إنشاء قوالبك الخاصة
- 📝 **إدارة القوالب**: إضافة، تعديل، حذف قوالب الرسائل المخصصة
- 🕐 **السجل الأخير**: حفظ تلقائي للأرقام التي راسلتها مؤخرًا
- 📱 **دعم واتساب المزدوج**: يعمل مع واتساب العادي وواتساب بزنس
- 🌙 **دعم المظاهر**: مظهر فاتح، داكن، ومظهر النظام
- 🌐 **ثنائي اللغة**: واجهة كاملة بالعربية والإنجليزية مع دعم RTL
- 🔒 **الخصوصية أولاً**: لا جمع للبيانات، كل شيء محلي
- 🚫 **بدون إعلانات**: مجاني بالكامل بدون إعلانات
- ⚡ **خفيف**: أقل من 50 ميجا، سريع وفعال

## 🚀 Getting Started / البدء

### Prerequisites / المتطلبات

```bash
Flutter SDK >= 3.9.2
Dart SDK >= 3.9.2
Android Studio / VS Code
```

### Installation / التثبيت

```bash
# 1. Clone the repository / استنساخ المستودع
git clone https://github.com/salehbagomri/quickchat.git

# 2. Navigate to project directory / الانتقال لمجلد المشروع
cd quickchat

# 3. Install dependencies / تثبيت التبعيات
flutter pub get

# 4. Generate code (Hive adapters) / توليد الكود
flutter pub run build_runner build --delete-conflicting-outputs

# 5. Run the app / تشغيل التطبيق
flutter run
```

## 🔨 Building / البناء

### Android APK
```bash
# Debug build / بناء تجريبي
flutter build apk --debug

# Release build / بناء إصدار
flutter build apk --release

# Split per ABI (smaller size) / تقسيم حسب المعمارية (حجم أصغر)
flutter build apk --release --split-per-abi
```

### Android App Bundle (Recommended for Play Store)
```bash
flutter build appbundle --release
```

### iOS
```bash
# iOS App / تطبيق iOS
flutter build ipa --release
```

### Output locations / مواقع الملفات الناتجة:
- APK: `build/app/outputs/flutter-apk/app-release.apk`
- AAB: `build/app/outputs/bundle/release/app-release.aab`
- IPA: `build/ios/ipa/*.ipa`

## 📦 Deployment / النشر

### Google Play Store
1. Follow the complete guide: `docs/FINAL_PUBLISH_GUIDE.md`
2. Or use quick guide: `docs/QUICK_PUBLISH_GUIDE.md`

4. Upload to Play Console
5. Submit for review

1. Configure Xcode signing
2. Build: `flutter build ipa --release`
3. Upload via Transporter or Xcode
4. Submit for review
