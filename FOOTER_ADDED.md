## 🎨 التصميم:

```dart
Widget _buildFooter(AppLocalizations l10n) {
  return Center(
    child: Padding(
      padding: const EdgeInsets.symmetric(vertical: 8),
      child: Text(
        l10n.madeWithLove,
        style: TextStyle(
          fontSize: 12,
          color: Colors.grey[600],
        ),
        textAlign: TextAlign.center,
      ),
    ),
  );
}
```
