## 🔄 سير العمل (Workflow):

### إضافة قالب جديد:
```
1. فتح الشاشة الرئيسية
2. الضغط على أيقونة القوالب 📋
3. الضغط على زر + (إضافة)
4. إدخال العنوان والرسالة
5. الضغط على "حفظ"
6. ✅ القالب جاهز للاستخدام!
```

### استخدام قالب:
```
1. الضغط على أيقونة القوالب 📋
2. اختيار القالب المطلوب
3. ✅ يتم ملء الرسالة تلقائياً
4. (اختياري) تعديل الرسالة
5. إرسال عبر واتساب
```

### تعديل قالب:
```
1. فتح شاشة القوالب
2. الضغط على ⋮ بجانب القالب
3. اختيار "تعديل"
4. تعديل العنوان أو الرسالة
5. الضغط على "حفظ"
6. ✅ تم التحديث!
```
