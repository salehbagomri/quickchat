# Changelog

All notable changes to QuickChat will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-09

### Added - الإضافات
- 🚀 Initial release / الإصدار الأول
- 📱 Open WhatsApp without saving contacts / فتح واتساب بدون حفظ جهات الاتصال
- 🌍 Country code picker with Yemen as default / اختيار رمز الدولة مع اليمن كافتراضي
- 💬 Optional message field / حقل رسالة اختياري
- 📜 Chat history with local storage / سجل المحادثات مع تخزين محلي
- 📋 Message templates (10 default + custom) / قوالب الرسائل (10 افتراضية + مخصصة)
- 🌙 Dark/Light/System theme support / دعم الوضع الداكن/الفاتح/النظام
- 🌐 Arabic and English localization with RTL support / العربية والإنجليزية مع دعم RTL
- ⚙️ Settings screen with theme and language options / شاشة إعدادات مع خيارات المظهر واللغة
- 🔒 Privacy Policy / سياسة الخصوصية
- 📧 Contact developer feature / ميزة التواصل مع المطور
- ⭐ Rate app option / خيار تقييم التطبيق
- 🎨 Beautiful UI with Material Design 3 / واجهة جميلة مع Material Design 3
- 🔄 Auto-detect system language on first launch / كشف تلقائي للغة النظام عند أول تشغيل

### Features - المميزات
- ✅ No data collection - complete privacy / لا يوجد جمع بيانات - خصوصية كاملة
- ✅ All data stored locally (Hive) / جميع البيانات مخزنة محلياً
- ✅ No internet required / لا يتطلب إنترنت
- ✅ Lightweight and fast / خفيف وسريع
- ✅ No ads / بدون إعلانات
- ✅ No tracking / بدون تتبع
- ✅ Open source friendly / صديق للمصادر المفتوحة

### Technical - تقني
- Built with Flutter 3.x
- Minimum Android version: 5.0 (API 21)
- Target Android version: 14 (API 34)
- Uses Material Design 3
- State management with BLoC pattern
- Local storage with Hive
- Internationalization with Flutter Intl
