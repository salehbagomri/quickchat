## 🔧 التعديل الوحيد:

### في `SettingsCubit`:

```dart
Future<void> changeLanguage(String languageCode) async {
  // 1. حفظ اللغة الجديدة
  await _preferencesService.setLanguage(languageCode);
  
  // 2. تحديث واجهة التطبيق
  emit(state.copyWith(locale: Locale(languageCode)));
  
  // 3. إعادة توليد القوالب تلقائياً باللغة الجديدة ✅
  try {
    await _templateService.regenerateDefaultTemplates(languageCode);
  } catch (e) {
    debugPrint('Error regenerating templates: $e');
  }
}
```

### خطوات المستخدم:

```
1. يفتح: الإعدادات ⚙️
2. يغير اللغة:
   ○ English → ● العربية
3. ✅ كل شيء يتغير:
   - واجهة التطبيق → عربية
   - القوالب → عربية (10 قوالب)
   
أو العكس:
   ● العربية → ○ English
   ✅ كل شيء إنجليزي (بما فيها القوالب)
```
