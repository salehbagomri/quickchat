## 🔧 التعديلات التقنية:

### 1️⃣ في `TemplateService`:

```dart
/// دالة جديدة لإعادة توليد القوالب
Future<void> regenerateDefaultTemplates(String languageCode) async {
  // 1. حذف جميع القوالب الحالية
  await clearAllTemplates();
  
  // 2. إضافة القوالب الجديدة باللغة المختارة
  final defaultTemplates = languageCode == 'ar' 
      ? _getArabicTemplates()   // 10 قوالب عربية
      : _getEnglishTemplates(); // 10 قوالب إنجليزية
  
  for (var template in defaultTemplates) {
    await addTemplate(
      title: template['title']!,
      message: template['message']!,
    );
  }
}
```

### 2️⃣ في `SettingsScreen`:

```dart
// قسم جديد في الإعدادات
Widget _buildTemplatesSection(...) {
  return Card(
    child: ListTile(
      leading: Icon(Icons.refresh),
      title: Text(l10n.regenerateTemplates),
      subtitle: Text(
        // يعرض اللغة التي ستُستخدم
        state.locale.languageCode == 'ar' 
            ? 'إعادة توليد القوالب بالعربية'
            : 'Regenerate templates in English',
      ),
      onTap: () => _showRegenerateTemplatesDialog(...),
    ),
  );
}
```

### 3️⃣ رسالة التأكيد:

```dart
Future<void> _showRegenerateTemplatesDialog(...) async {
  // 1. عرض رسالة تأكيد
  final confirmed = await showDialog<bool>(...);
  
  if (confirmed == true) {
    // 2. عرض مؤشر تحميل
    showDialog(...CircularProgressIndicator...);
    
    // 3. إعادة توليد القوالب
    await templateService.regenerateDefaultTemplates(
      state.locale.languageCode
    );
    
    // 4. إغلاق المؤشر وعرض رسالة نجاح
    Navigator.pop(context);
    ScaffoldMessenger.showSnackBar(...);
  }
}
```

## 🎯 سيناريوهات الاستخدام:

### سيناريو 1: مستخدم عربي يريد قوالب إنجليزية

```
1. يفتح التطبيق (قوالب عربية افتراضياً)
2. الإعدادات → اللغة → English
3. الإعدادات → إعادة توليد القوالب
4. يضغط "Confirm"
5. ✅ الآن لديه 10 قوالب إنجليزية!
```

### سيناريو 2: مستخدم إنجليزي يريد قوالب عربية

```
1. يفتح التطبيق (قوالب إنجليزية افتراضياً)
2. Settings → Language → العربية
3. الإعدادات → إعادة توليد القوالب
4. يضغط "تأكيد"
5. ✅ الآن لديه 10 قوالب عربية!
```

### سيناريو 3: مستخدم يريد تحديث القوالب

```
1. لديه قوالب قديمة (مخصصة أو افتراضية)
2. الإعدادات → إعادة توليد القوالب
3. رسالة تحذير: "سيتم حذف الكل..."
4. يضغط "تأكيد"
5. ✅ قوالب جديدة نظيفة بلغته الحالية!
```
